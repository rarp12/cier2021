Diligenciar un Instrumento

Esta opción  abre una pantalla, la cual contiene una pestaña por cada modulo que conforma el instrumento de digitación.  Se puede reconocer el modulo en el cual se encuentra ubicado durante la digitación, porque este es resaltado en color azul. Adicional para ir ubicándose dentro del instrumento, cada módulo está compuesto por secciones que se diferencian por códigos, al igual que sus preguntas.

 

 

 

MÓDULO 1: INFORMACIÓN GENERAL

 

Contiene la información de descripción básica del predio.  Está compuesto por las secciones de datos generales de la encuesta,  datos del establecimiento o complejo educativo, ubicación geográfica y cobertura y usuarios.

DATOS GENERALES DE LA ENCUESTA

 

En esta sección  todos los campos son obligatorios.

Validación: El sistema valida que el usuario no ingrese el mismo número de boleta censal para dos instrumentos del mismo predio. 

 

DATOS DEL ESTABLECIMIENTO O COMPLEJO EDUCATIVO

Es esta sección se debe digitar la información correspondiente al predio,  establecimiento Principal y el establecimiento secundario. Para el caso en el que el predio no exista en el sistema se tiene la opción de adicionarlo durante la creación del instrumento.

Si en la encuesta física no se digitaron códigos de los establecimientos, el sistema brinda la posibilidad de buscar un predio, para esto se debe dar clic en el botón con el icono de una lupa  y este  abrirá una ventana emergente para realizar la búsqueda del predio a través de listas dependientes.

  

 

Si no es posible encontrar el predio, es necesario crearlo, para esto se utiliza el botón  de Crear Predio, el cual abrirá  una ventana emergente que permitirá la creación de un predio, seleccionando la información por los niveles de DPA anteriores al predio.

MODULO 3 - INFORMACIÓN DEL EDIFICIO ESCOLAR

 

Este módulo permite el registro de toda la información de edificios del predio.

Un instrumento puede tener múltiples edificios, por defecto el sistema carga una tabla con un  elemento edificio vacío, para que sea diligenciado.

 

 

 

 Si se desea adicionar un nuevo edificio se debe presionar el botón Agregar Edificio, esta opción adicionará un nuevo edificio al final de tabla para que sea diligenciado.

 

Para editar un edificio, se debe dar clic sobre el botón con el  icono de un lápiz  , este abrirá una ventana emergente en la que será posible editar la información.

 

 

Una vez se haya digitado la información correspondiente, debe presionar la opción guardar en la parte inferior derecha de la ventana. Si no lo hace la información no será guardada en el sistema.

Para guardar el edificio como mínimo se debe digitar la pregunta EDIFICIO NO, una vez hecho esto se puede proceder a guardar la información adicional.

 

Si se desea eliminar un edificio, se debe presionar el botón con el icono de una papelera  , esta opción lanzara una ventana de confirmación en la cual podrá cancelar o confirmar la acción. Se debe ser muy cuidadoso con esta opción porque si realiza la eliminación  del sistema no será posible recuperar la información eliminada.

 

 

 

 

MODULO 4 - ESPACIOS ESCOLARES

 

INFORMACION PARA ESPACIOS ESCOLARES

 

Este módulo permite el registro de toda la información de espacios del predio.

Un instrumento puede tener múltiples espacios por edificio,  por defecto el sistema carga una tabla con un  elemento espacio vacío, para que sea diligenciado.

 

 

 Si se desea adicionar un nuevo espacio se debe presionar el botón Agregar espacio , esta opción adicionará un nuevo espacio al final de tabla para que sea diligenciado.

 

Para editar un espacio, se debe dar clic sobre el botón con el icono de un lápiz  , este abrirá una ventana emergente en la que será posible editar la información.

  

Una vez se haya digitado la información correspondiente, debe presionar la opción guardar en la parte inferior derecha de la ventana. Si no lo hace la información no será guardada en el sistema.

Para realizar el almacenamiento de un espacio se debe digitar el campo ID- Espacio Escolar N° (con valor numérico) y Edificio o Exterior N° (con valor numérico antecedido de una letra E o EX para indicar si es Edificio o Edificio Exterior).

Si se desea eliminar el espacio de un edificio, se debe presionar el botón con el icono de una papelera , esta opción lanzará una ventana de confirmación en la cual podrá cancelar o confirmar la acción. Se debe ser muy cuidadoso con esta opción porque se realiza la eliminación en el sistema y no es posible recuperar la información eliminada.

 Si se desea adicionar un nuevo plano se debe presionar el botón Agregar Croquis , esta opción adicionará un nuevo edificio al final de tabla para que sea diligenciado.

 

Para editar un plano, se debe dar clic sobre el botón con el icono de un lápiz  , este abrirá una ventana emergente en la que será posible editar la información.


Una vez se haya digitado la información correspondiente, debe presionar la opción guardar en la parte inferior derecha de la ventana. Si no lo hace la información no será guardada en el sistema.

 

Si se desea eliminar un edificio, se debe presionar el botón con el icono de una papelera  , esta opción lanzara una ventana de confirmación en la cual podrá cancelar o confirmar la acción. Se debe ser muy cuidadoso con esta opción porque si realiza la eliminación del sistema  no será  posible recuperar la información eliminada.

 

 

 

A1 - ANEXO INFORMATIVO

 

Este anexo permite registrar los ambientes similares y las observaciones complementarias.

AMBIENTES SIMILARES

 

Este al igual que edificios y establecimientos es representado con un elemento múltiple,  por defecto el sistema carga una tabla con un  elemento de ambientes similares vacío, para que sea diligenciado.

 

  

Si se desea adicionar un nuevo ambiente similar se debe presionar el botón, esta opción adicionará un nuevo ambiente similar al final de tabla para que sea diligenciado.

Para editar un ambiente similar, se debe dar clic sobre el botón con el  icono de un lápiz , este abrirá una ventana emergente en la que será posible editar la información.

 

 

En este formulario, se digita en la primera línea el ambiente el cual se va tomar como referencia para los ambientes similares, este sería el ambiente principal. En los campos de abajo se van listando todos los ambientes que contengan las mismas características.

Una vez diligenciada la información se almacena con el botón guardar ubicado en la parte inferior de la ventana emergente, en caso contrario se hace clic en el botón cancelar lo cual cerrara la ventana sin realizar ningún registro de operación.

 

Si se desea eliminar un AMBIENTE SIMILAR, se debe presionar el botón con el  icono de una papelera , esta opción lanzará una ventana de confirmación en la cual podrá cancelar o confirmar la acción. Se debe ser muy cuidadoso con esta opción porque si realiza la eliminación del sistema no será posible recuperar la información eliminada.

  

Adicional a esto para confirmar la correcta terminación de la operación de eliminación se muestra un mensaje confirmando el éxito de la operación en la ventana principal del instrumento.

 

  

 

 

OBSERVACIONES COMPLEMENTARIAS

 

Las observaciones tienen un comportamiento similar al resto de los elemento múltiples del instrumento, inicialmente se carga por defecto uno para ser editado.

   

El formulario de edición de las observaciones complementarias es el siguiente:

 

 

 

Una vez se digiten los datos, se confirma la operación con el botón Guardar, es de aclarar que  en caso de no digitar ninguna respuesta el elemento no será guardado. En caso que no se quiera guardar el elemento se hace clic en el botón Cancelar.

 

El proceso de eliminación es similar a los anteriores, se hace clic en el botón con el icono de una papelera  . Posteriormente se desplegará la confirmación de la operación, si se quiere eliminar se hace clic en el botón de Aceptar, en caso contrario en  Cancelar.

 

 

Adicional a esto para confirmar la correcta terminación de la operación de eliminación se muestra un mensaje confirmando el éxito de la operación en la ventana principal del instrumento.

Posteriormente se desplegará la confirmación de la operación, si se quiere eliminar se hace clic en el botón de Aceptar, en caso contrario en el botón de  cancelar, se debe tener en cuenta que el listado de elementos en el instrumento es único, por tanto si se eliminan en este módulo, el resultado se verá reflejado en todos los módulos que utilicen el listado de elementos de tipo edificio.

CONSULTAR UN INSTRUMENTO

 

En esta opción de la aplicación, se permite realizar la búsqueda de un instrumento por unas condiciones dadas inicialmente por el usuario, esto arrojará un listado de instrumentos a los cuales se podrá acceder para realizar su consulta o modificación.

Para acceder a esta opción se hace clic en el menú de digitación

 

  

 

 

En el menú lateral que se despliega, de hace clic en consultar Instrumento 

 

 

El formulario que se muestra, está compuesto de dos secciones, la de filtros de consulta y la de resultados.

 

Filtros de Consulta: Muestra las diferentes opciones de filtrado de los instrumentos, en este caso se tiene los geográficos que permiten filtrar por los datos de la división político administrativa junto con el código del establecimiento, y los filtros básicos en los cuales se puede filtrar por periodo, boleta censal y estado, una vez se seleccionen los filtros deseados se hace clic en consultar, además de esta opción se pueden emitir múltiples fichas al tiempo, solo se debe chequear la casilla que aparece al principio del listado y se disparara el proceso de emisión en lote, esto se hace haciendo clic en el botón Emitir.

 

 

 

Posterior a esto se cargara un listado con los instrumentos que cumplen con los criterios de búsqueda seleccionados por el usuario.

 

 

 

En el listado se muestran los datos básicos correspondientes a los instrumentos como son: número de boleta censal, código y nombre del establecimiento, código y nombre de la sede y código y nombre del predio.

 

Dentro de la consulta se pueden realizar las siguientes operaciones indicadas por los iconos ubicados en las columnas ubicadas al lado derecho de la pantalla. Algunas de las operaciones son: modificación, impresión en PDF, ver mapa del predio entre otras.

MODIFICAR UN INSTRUMENTO

 

Esta opción se puede modificar la información de las respuestas digitadas del instrumento, para realizar la modificación, de debe primero consultar el instrumento luego se hace clic en el icono de modificación. En una nueva ventana se cargarán todas las respuestas previamente digitadas, como usuario con los permisos suficientes, podrá modificar todas las respuestas.

Para confirmar esta operación se debe hacer clic en el botón Guardar y toda la información nueva digitada será almacenada.

  

Para modificar un instrumento solo puede hacerlo en estado nuevo, de otra manera no es posible realizar esta operación.

CAMBIAR DE ESTADO DE UN INSTRUMENTO

 

Anular Instrumento:

Esta opción como su nombre lo indica realiza la anulación de un instrumento,  esta anulación implica que la boleta no será tenida en cuenta durante la posterior generación de indicadores. Para proceder con esta, se debe hacer clic en el botón anular ubicado en la parte superior derecha del formulario del instrumento.

  

Una vez se presiona el botón anular, se desplegará una ventana en la cual se debe digitar una observación, esta es de tipo obligatorio.

 

  

 

Una vez se digite la observación y se haga clic en aceptar, se mostrará un mensaje de confirmación de la operación.

 

 

Para completar y volver al instrumento se hace clic en el botón de Cancelar O en el icono de cerrar ubicado en la parte superior de la ventana.

 

Emitir Instrumentos

Esta opción permite la emisión individual o en bloque de instrumentos, para hacer uso de ella se deben seleccionar los instrumentos que se desee emitir marcando las casillas ubicadas en la primera columna de la tabla de resultado de la consulta básica.

 

 

 

 

 

 


Una vez se seleccionan los instrumentos y se hace clic en el botón de Emitir se pueden verificar los resultados en la parte superior de la pantalla.

 

 

El resultado consiste en un archivo txt donde se especifican las respuestas que se deben completar para poder emitir el instrumento.
