Create First Session User

To enter a country for the first time we must create what we will denominate Facility Super User. This user will have all minimum administrative permissions related to the creation of users and roles. This users default role will be the system manager and will be addressed as Manager. To create this user click on the button, it will display this fill-out form:

All fields marked with * are mandatory.

Once the form is filled out, click on Save and there will be the first session user in the country. If for any reason the authentication credential is forgotten, the form must be filled out once again and this will override the previous Super User data.
