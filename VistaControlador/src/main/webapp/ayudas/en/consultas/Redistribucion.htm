This window allows us to evaluate the use of each the institutional lands and define the places for students on each shift, keeping in mind the current number of students and the maximum number they can hold in an institutional land according to the square meters per classroom available.  

For aggregated levels, you can see the number of lands that are using more than they can (crowded) or overcrowded as well as those which are using less than they can; thus it’s possible to analyze the possibility to balance the students per school according to its capacity.
