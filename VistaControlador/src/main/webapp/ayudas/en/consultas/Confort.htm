Comfort

 

Objectives

This query allows the user to quantify the number of spaces in each building of each institutional land according to the state of the natural and artificial lightning or acoustics functionality and natural ventilation.

It will also be possible to see the number of spaces with thermal insulation on the floors, roof, Windows or walls.

On the other hand its possible to analyze the state of the showers, urinals, sinks and toilettes as well as quantifying its number according to the state they are in, in order to define the proportions of the sanitary tools.

 

Features

·         Natural lightning condition: this variable allows the user to know the use of natural lightning in each of the schools areas, according to the following quantitative scale from one (1) to four (4):

1.       There is no lightning.

2.       Insufficient lightning.

3.       Sufficient lightning.

4.       Normal lightning.

 

·         Artificial lightning condition: this variable allows the user to know the use of artificial lightning in each of the schools areas, according to the following quantitative scale from one (1) to four (4):

1.       There is no lightning or the system is out of service or doesnt work.

2.       The lightning is insufficient, its partially or completely installed but requires maintenance.

3.       The lightning is sufficient and works appropriately. 

4.       The lightning is normal and works correctly. 

 

·         Natural Ventilation Condition: this variable allows the user to know the use of natural ventilation in each of the schools areas, according to the following quantitative scale from one (1) to four (4):

1.       There is no ventilation.

2.       Insufficient ventilation.

3.       Sufficient ventilation.

4.       Normal ventilation.

 

·         Acoustics Functionality Condition: this variable allows the user to know the use of sound propagation in each of the schools areas, according to the following quantitative scale from one (1) to four (4):

1.       Acoustics looks affected in a HIGH level by outdoor noises. 

2.       Acoustics looks affected in a MEDIUM level by outdoor noises. 

3.       Acoustics looks affected in a LOW level by outdoor noises. 

4.       Acoustics is normal, there is no interruption from outdoor noises.

 

·         Condition of showers, sinks, urinals and toilets: this variable allows the user to know the condition of showers, sinks, urinals and toilets, as well as the corrective action to minimize the impact, according to the following quantitative scale from one (1) to four (4):

1.       Between 50 and 100% of the appliances installed are out of service, dont work or because of their condition need to be changed.

2.       Less than 50% of the appliances need to be changed due to their condition and/or are partially installed. 

3.       All of the sanitary appliances work but require preventive maintenance. 

4.       All of the sanitary appliances work normally and show appropriate maintenance.
