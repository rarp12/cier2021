MODULE 3 – SCHOOL BUILDING INFORMATION

 

This module allows the register of all the building’s property information.

An instrument can have multiple buildings, in order to fill it out the system loads a table with an element Empty Building by default.

 

 

If the user wants to add a new building you should press the  button, this option will add a new building at the end of the table to fill it out.

 

In order to edit a building click on the icon, this will open an emerging window in which you will be able to edit the information. 

 

Once you have entered correspondent information, you should press the Save option which is located in the lower-right of the window. If not, the information won’t be saved the system.

In order to save the building you should at least enter the question “NO BUILDING”, Once this is done proceed to save the additional information.

 

If you want to erase a building you should press the icon, this option will launch a confirmation window where you will be able to cancel or confirm the action. Be very careful with this option because the system erasing begins, and it’s not possible to recover the erased information.

SPECIFIC DIAGRAM FOR EDUCATIONAL FACILITIES

 

This module allows the register of all the diagrams’ information and educational facilities. This last one just like the rest of buildings and establishments which is represented with a multiple element, the system loads a table with an empty space diagram to be filled out by default.

 

 

 

If you want to add a new diagram press the button, this option will be added to a new building at the end of the table to be filled out. 

In order to edit a diagram click on the icon, this will pop-up a window where all the information will be able to be edited.

Once all the correspondent information is entered press the save option in the lower-right part of the window. If not all the information won’t be saved in the system.

SIMILAR ENVIRONMENTS

 

This last one just like the buildings and establishments is represented with a multiple element; the system loads a table with an empty Similar Environment element to be filled out. 

  

If you want to add a new Similar Environment press the button, this option will add a new Similar Environment at the end of the table to be filled out.

In order to edit a Similar Environment click on the icon, this will pop up a window where you will be able to edit all the information. 

 

Enter in the first line the environment which will be used to take as a reference for similar environments on this form; lists will be made that have the same main environment’s characteristics.

Still watching the saved information with the save button located in the lower part of the popped-up window,  On the contrary click in the cancel button which will close the window if a new register is made with no operation register.

 

If you want to erase a Similar Environment press the icon, this option will launch a confirmation window where you will be able to cancel or confirm the action. Be careful with this option because the erasing will be made in the system and is not possible to recover the erased information.

  

Additionally in order to confirm the finish of the erasing operation a message is shown confirming the operation success on the instrument’s main window.

CONSULTING AN INSTRUMENT

 

In this application’s option, it is allowed to make the search of an instrument with some conditions given at first by the user, this will give a list of instruments which you will Access to make the consul or modification.

In order to Access this option click on the data entry menu

 

  

 

Click on the side-menu that deploys in the consulting instrument

 

 

The form that shows up has two sections, the consulting filters and results.

 

Consulting filters:  Show the different options of instruments filtering, in this case you have the geographic data which allows  filtering  using data of the Politic-Administrative Division alongside the establishment code, and the basic  filters where you can filter per period, and the basics where you can listen the census and state of this, once the filters are selected the click on the consulting, besides this option can show multiple pieces,  just check the box which appears in the beginning of the list,  and will shoot the process’ issue in the box that appears at the beginning of the list and will shoot the lot’s emission process, this is done doing click on the button EMIT.

 

 

Previously to this a list will load with the instruments that have the same search criteria selected by the user. 

 

 

On the list the basic data corresponding to the instruments are shown:  census’s ticket number, code and establishment name, code and name of the branch, and the name will be.

 

With the consulting you can also make the following indicated operations by the located icons in the located columns in the right-side of the screen. Some of the operations are: modify PDF printing, watch Property Map among others.

Modify instrument.

Print PDF.

See the map.

Pass to a New State

Create news


Watch pictures

 


Watch the diagrams

 

In this operation you should take into account the following:

Modify the Instrument: the instruments which are in new state can be modified.

Pass to a New State: The instruments that can be passed to new be the instruments which have the state of null and emitted.

Create Novelty: This option is only available for instruments in Null or Emitted state, and should take into account that a copy will be made with the last emitted instrument for the corresponding property, in case of not having, it will create a new one.

Besides these operations, in the first line it shows a color box indicating the instrument’s state, the colors convention is: Blue (new), Green (emitted) and red (null).

MODIFY AN INSTRUMENT

 

This option can be modified the information of the instrument’s entered answers, to make the modification, you should first consult the instrument then click on the modification icon. On a new window all the answers will be entered as user with the enough privileges you will modify all the answers.

To confirm this operation you should click on the save button and all the new entered information will be stored.

  

To modify an instrument you can just do it on new state, it’s not possible to do it in another way.

 

CHANGING THE STATE OF AN INSTRUMENT

 

Anulating instrument:

This option just like its name indicates nulls an instrument, to proceed with this, click in the null button located in the upper-right part of the instrument’s form.

  

Once the annulation button is pressed, a window will be deployed where you have to type in an observation, this is a must. 

  

 

Once the observation is entered and clicked on accepting a confirmation message will pop-up.
