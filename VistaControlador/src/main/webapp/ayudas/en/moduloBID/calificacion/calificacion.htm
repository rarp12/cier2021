HELP WINDOW COUNTRIES’ GRADE

This module gives the user the consulting grade option per country, in the BID’S managing module.

In order to access the Countries’ grade option, Access the BID’s module in the application’s main menu.


Once inside the module in the side-menu click on the Grade option.

 

 

 

 

 

 

 


In the option, the next window will pop-up

 


Located in the upper part of the screen, select in as first step the period which will be consulted the data, once selected this, click on the CONSULT button.

Next the list with the corresponding information of the assigned grade per country will pop up.


Additionally you have the option to hide or show the columns the first time that you consult all the available columns will show up, in order to hide or show select them in the group located in the upper part of the results’ table and click on the mark.

 


 

In the inform’s columns there is available the option of exporting to PDF the BID inform per country, this is done in the button located in the last in form’s column, once done click and download a PDF document with the detailed BID’s inform.

 


Another functionality of exporting to Excel the generated inform (XLS), for this click on the export button located in the upper-right part of the results’ table.
