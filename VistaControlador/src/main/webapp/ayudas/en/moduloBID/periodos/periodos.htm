1.1           Create periods

 

This module gives the user the create option, modify and erase the periods which are going to be worked with in the managing BIDs module

In order to access the create period option, we should access the BID module in the applications main menu.

 


Once in the module in the side menu click on the Periods Admin option.

 

 

 

 

 

 

Once in the option, the following window will pop-up which have the created periods in the application.

 

 

 

 

 

To proceed with the creation, click on the Add Period button.

 

The periods creation form is the following:

 

 

 

 

 

 

The data to create a period are:

·         Code: This is an alphanumeric type of field and is required for the period creation.

·         Description: This is an alphanumeric field and is not required for the period creation.

Take into account that you cant create periods with the same code.

To complete the creation operation, click on the safe button.

1.2           Modify periods

 

The period modification option is available in the consulting periods table, to proceed with this you have to click on the edit button  from the line corresponding to the period.

When clicking on the button, the line will activate in Edition mode, allowing modifying the registered information in the period.

 

 

 

 

Once all the data are modified, click on the mark at the end of the line to confirm the operation in the case of not wanting to proceed with the modification, click on the cross.

1.3           Erasing Periods

 

The erasing periods option is available in the table where consulting periods is, to proceed with this click on the  Erase button.

 

Take into account that if a period has registered information it wont be able to be erased.
