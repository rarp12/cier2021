Objective

This query allows the user to identify the relationships between the variables from different scopes, according to the preselected options in order to prioritize decisions on educational infrastructure n institutional lands.

This query aims to match variables from different scopes and facilitate the identification and prioritization of problems. It is important to remember that this query is only enabled for the level of institutional land.
